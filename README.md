# SCiXus: Production RAG Assistant Case Study

A case study of SCiXus, a retrieval-augmented chatbot I designed, built, and
deployed to answer ERP support questions for retail staff, live in Microsoft
Teams and grounded in ~18,000 chunks of internal documentation.

The underlying code and corpus are proprietary and remain private; this repo
contains only the writeup.

- **[CASE_STUDY.md](./CASE_STUDY.md)**: problem, architecture, key decisions, results, limitations
- **[ARCHITECTURE.md](./ARCHITECTURE.md)**: module-by-module technical deep dive

I was the sole architect and engineer on this project end to end.
